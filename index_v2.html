About The Site

MSc Project

This webpage is part of the deliverables for the 2024/2025 MSc Project at the College of Science & Engineering, University of Glasgow.

This is an application-oriented MSc project under Theme P: "Bringing the field to your desktop: Accessibility-focused spatial analysis and visualisation of GES field course locations and activities." The study area and research focus have been slightly modified based on personal interest, with changes approved after discussion with supervisors. The webpage and its web map component were developed with the assistance of W3Schools and Mapbox tutorial examples. The code has been modified where necessary.

Research Methodology

Data Collection

Data used in this project were fully collected online from a variety of sources including government mapping agency, government open portals, open geodatabase, etc. Below is the summary of the key data used in the project:

  • Road network
  • Healthcare facilities
  • Population data
  • Neighbourhood Blocks

Software & Application

A range of software applications was used during the development of this project:

  • QGIS for spatial data processing
  • ESRI ArcGIS Pro
  • Visual Studio Code
  • Mapbox GL JS for interactive visualization
